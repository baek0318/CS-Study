## 업데이트 일관성
여러사람이 하나의 컬럼을 동시에 업데이트할려는 경우 비일관성의 문제가 발생할 수 있다  
이것을 가지고 `쓰기 충돌`이라고 이야기한다
### 해결방법
1. `비관적 방법`
   - 충돌이 발생하는 것을 방지하는 방식으로 작동하는 것
   - 대표적인 방법 : 쓰기 잠금
2. `낙관적 방법`
   - 충돌이 발생하도록 놔두고 충돌이 발생하면 해결하는 것
   - 대표적인 방법 : 조건적 업데이트
